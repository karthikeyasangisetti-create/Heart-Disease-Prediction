# ❤️ Heart Disease Prediction Using Machine Learning

## Overview

This project focuses on building and evaluating machine learning models to predict the likelihood of heart disease based on patient health information. The objective was to preprocess the dataset, train multiple classification models, compare their performance, and identify the most accurate model.

The project was completed as part of the **AI & Machine Learning Internship** at Pluto Academy.

---

## Project Objectives

- Load and preprocess the dataset
- Perform feature analysis
- Train multiple machine learning models
- Compare model performance using evaluation metrics
- Select the best-performing model

---

## Dataset

- **Dataset:** Heart Disease Prediction Dataset
- **Source:** Kaggle
- **Link:** https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

---

## Tools and Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

---

## Machine Learning Models

The following classification algorithms were implemented:

- Logistic Regression
- Random Forest Classifier
- K-Nearest Neighbors (KNN)

Each model was trained using the same training dataset and evaluated using the same testing dataset for a fair comparison.

---

## Project Workflow

### 1. Data Loading and Exploration

The dataset was loaded into Python and explored to understand its structure, identify missing values, and examine the available features.

### 2. Data Preprocessing

The preprocessing stage included:
- Handling missing values
- Feature selection
- Splitting the dataset into training and testing sets
- Preparing the data for machine learning

### 3. Model Training

Three different machine learning algorithms were trained on the dataset to predict heart disease.

### 4. Model Evaluation

The trained models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

A comparison table was created to compare the performance of all models.

---

## Results

The models achieved strong prediction performance, with one model outperforming the others based on the evaluation metrics.

The comparison process demonstrated the importance of evaluating multiple algorithms instead of relying on a single model.

---

## Conclusion

This project strengthened my understanding of the complete machine learning workflow, including preprocessing, model training, evaluation, and performance comparison. It also provided practical experience in selecting an appropriate model based on quantitative evaluation metrics rather than assumptions.
